# Neosrc

A refined UX for GitHub and other Git platforms.

Neosrc is a drop-in replacement for github.com that focuses on making code review faster and more pleasant. It is fully open source.

## Project status

Early development. Many things are rough or missing.

## How it works

Neosrc authenticates with your GitHub account and serves a reimagined PR review interface.
The URL scheme mirrors GitHub's: `/{owner}/{repo}/pull/{number}`.
